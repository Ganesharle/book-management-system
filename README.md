# Book Management System

## Features

- View books
- Add books
- Edit books
- Delete books
- Search books
- Filter by genre

## Tech Stack

- React
- Axios
- MockAPI
- Vite

## Setup Instructions

1. Clone repository

git clone https://github.com/Ganesharle/book-management-system.git

2. Install dependencies

npm install

3. Run project

npm run dev
